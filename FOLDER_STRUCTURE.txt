                         QUICK START COMMANDS
════════════════════════════════════════════════════════════════════════════

1️⃣  LOCAL DEVELOPMENT
    Backend:    cd backend && python main.py
    Frontend:   cd frontend/qtc-spark && npm run dev

2️⃣  PRODUCTION DEPLOYMENT (First Time)
    cd backend && chmod +x setup-gcp.sh && ./setup-gcp.sh YOUR_PROJECT_ID

3️⃣  CONFIGURE & DEPLOY
    cd backend/terraform
    cp terraform.tfvars.example terraform.tfvars
    terraform init && terraform apply

4️⃣  ENABLE CI/CD
    Option A (GCP):  gcloud builds submit
    Option B (GitHub): Add secrets, push to main
